# Prompt Stitch - Not Found (404)

## Contexto

Tela de fallback para rota inexistente.

## Estrutura obrigatoria

- Codigo 404 em destaque.
- Mensagem curta explicativa.
- Link/botao para voltar ao inicio.

## Prompt final para Stitch

Redesenhe a tela 404 mantendo estrutura simples: codigo, mensagem e retorno para pagina inicial. Nao alterar comportamento de navegacao.

Objetivo visual: pagina amigavel, clara e consistente com a identidade "Saude acolhedora". Evitar linguagem fria ou tecnica.

Incluir hierarquia limpa e CTA de retorno evidente para reduzir friccao do usuario.
