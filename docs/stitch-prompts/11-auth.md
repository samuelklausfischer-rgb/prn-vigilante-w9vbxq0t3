# Prompt Stitch - Auth

## Contexto

Tela de login do sistema, com foco em acesso rapido e confiavel para operadores.

## Estrutura obrigatoria

- Tela centralizada com card de autenticacao.
- Logo institucional.
- Titulo, subtitulo, campos email/senha, botao de submit.
- Texto auxiliar de uso restrito.

## Prompt final para Stitch

Redesenhe a tela de autenticacao mantendo todos os elementos funcionais atuais: logo, campos, botao de acesso e textos de apoio. Nao alterar fluxo de login.

Objetivo visual: transmitir confianca e acolhimento sem perder sobriedade institucional. A tela deve ser limpa, amigavel e clara para uso recorrente.

Aplicar identidade "Saude acolhedora" com foco em legibilidade, estados de foco visiveis e CTA principal evidente.

Garantir boa experiencia em desktop e mobile.
