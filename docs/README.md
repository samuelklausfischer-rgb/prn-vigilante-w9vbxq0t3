# Documentacao do PRN-Vigilante

Esta pasta concentra a documentacao tecnica e operacional do projeto.

## Estrutura

- `architecture/`: arquitetura de sistema, banco e contratos.
- `operations/`: checklists e guias operacionais de deploy/homologacao.
- `planning/`: planos de migracao, execucao e organizacao.
- `migrations/`: guias e scripts de contexto para migracoes.
- `analysis/`: relatorios tecnicos e analises.
- `presentations/`: materiais visuais (HTML/apresentacoes).

## Nota

Arquivos temporarios, rascunhos e artefatos nao produtivos foram movidos para a pasta `lixo/` na raiz do projeto para revisao posterior.
